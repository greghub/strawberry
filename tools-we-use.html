CSS Transitions

Slow connections. You lose visitors if the website loads longer than 3 seconds.

Your website has to work well on any connection. A user is not connected to wifi or a high-speed connection. There are cases when they use LTE, 3G, EDGE or even 56k modems. We make sure that it load as fast as possible on any connection. We use Slowy (http://slowyapp.com/, http://mschrag.github.io/) to test the website on many connections and to make the loading time minimal.

LLJS : Low-Level JavaScript

LLJS is a typed dialect of JavaScript that offers a C-like type system with manual memory management. It compiles to JavaScript and lets you write memory-efficient and GC pause-free code less painfully, in short, LLJS is the bastard child of JavaScript and C.
http://lljs.org/
